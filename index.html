JNiosEmu is a development environment and emulator with the purpose of making it easy to learn programming in assembler. Assemble your source with a single button click, immediately see how values change in a register or memory. All this without any prior knowledge of assembler programming or complex tool chains.

The emulated system is a subset of the Nios II processor from Altera.
